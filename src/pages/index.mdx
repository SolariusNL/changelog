## Framework v0.10.0 {{  date: "2024-01-25T00:00Z" }}

Framework v0.10.0 is a major release that adds new features, changes, and bug fixes. Framework has also been incremented to alpha status.

### <SparkleIcon /> New Features

- **Sound identification** - We've added a sound identification system that'll automatically identify sounds uploaded to the platform. This is for our new copyright enforcement policy.
- **Add new TOTP login flow** - We've made it easier to login with TOTP by building a new login flow.
- **New Solarius branding** - Solarius has undergone a rebranding to better reflect our vision and mission.
- **Add Redis servers** - We've added a Redis management system to the platform to give you more freedom over creations on Framework.
- **New landing page** - The landing page has been overhauled for a better user experience and to reflect our mission even better.
- **Region blocking** - Framework now supports region blocking for administrators and self-service.
- **Location details** - Framework now supports location details for administrators and self-service.
- **License keys** - Soon, Framework will become a paid product (for business customers). We've added license key support to the platform to prepare for this.
- **Protected users** - We've added a new protected user system to Framework to protect users from abuse.
- **Badges** - Framework now has a badge system for users to showcase their achievements and contributions to the platform.
- **Decals** - Users can now upload and manage decals on Framework.
- **SSO (Single Sign-On)** - Framework now supports SSO for administrators.
- **Punishment evidence** - Staff can now add evidence to punishments to confirm their validity, and to provide context.
- **Home redesign** - We've redesigned the home page to be more consistent with the rest of the platform, and to be more informative.

### <HiOutlineCog /> Changes

- **New launching prompt** - We've added a new launching prompt to be compliant with our new design standards.
- **Game view updates** - We've updated the game view to be faster, and to match our new design standards.

### <HiOutlineExclamation /> Fixes

- We fixed an issue that would show team funds (even when they're private) in the `getServerSideProps` response on team pages.
- We fixed a race condition with sound license identification.
- Originally, profile links didn't validate that the input included a valid URI scheme. This has been fixed.
- Fixed Dicebear avatars not being generated correctly.
- Fix impersonation banners on light theme.
- Seen notifications are no longer displayed in the administration interface.
- Fixed the country list having incorrect country codes.
- Fix an exploit that allowed users to provide negative values as Ticket amounts, resulting in their balance being increased.
- Fixed an issue that prevented OAuth2 applications from being deleted when there are still active tokens.

We hope you enjoy this release. You can [view the full release log here](https://github.com/SolariusNL/framework/releases/tag/0.10.0). Thank you for using Framework!

We hope you enjoy this release. You can [view the full release log here](https://github.com/SolariusNL/framework/releases/tag/0.9.0). Cheers!

## Framework v0.7.4 {{  date: "2023-05-28T00:00Z" }}

Framework v0.7.4 is a minor release that includes various new features, changes, and bug fixes.

### <SparkleIcon /> New Features

- **Bits** - Bits are a new currency that can be used to purchase items on the platform. Bits will be replacing daily prizes, and you will receive bits for daily logins.
- **Domains** - Verify and utilize your own domain for our soon-to-be-released Wiki and Status software-as-a-service products.
- **Chat notifications** - Hear a sound when you receive a new message in chat.
- **Delete messages** - Delete your own messages in a conversation.
- **Transaction date sort** - Sort transactions by specific date ranges.
- **Chat bubble colors** - Change the color of your chat bubbles.
- **Link safety** - You'll be prompted when clicking on external links to verify intent.
- **Verification** - Verified users will have a blue checkmark next to their name to indicate authenticity.

### <HiOutlineCog /> Changes

- **Client download platforms** - Updated client download platforms to include comprehensive Linux support.
- **Chat rate limits** - Increased chat rate limits to allow for more messages to be sent in a shorter period of time.
- **Navigation icons** - Updated navigation icons to be more consistent.

### <HiOutlineExclamation /> Fixes

- **Underlying chat infrastructure** - We've made some changes to the underlying chat infrastructure to improve performance and reliability.
- **OAuth search** - Fixed an issue preventing users from searching for OAuth applications.
- **Truncation in chat flyout** - Fixed an issue where long messages would be incorrectly truncated in the chat flyout.
- **Deleting sessions on lock** - Fixed an issue where sessions would be deleted when locking your account. This was not intended behavior.

We hope you enjoy this release. Next update is scheduled for **June 30th, 2023**. [See full release log](https://invent.solarius.me/Soodam.re/framework/-/releases/0.7.4).
